iPhone/iPad에서는 Safari 브라우저로 접속해야 홈 화면에 추가할 수 있습니다.
1
Safari로 이 페이지 열기
Chrome이나 다른 브라우저가 아닌 Safari로 접속하세요. 주소창에 이 페이지 URL을 입력하거나, Safari에서 QR 코드를 스캔하세요.
Safari 앱 → 주소창에 URL 입력
2
하단 공유 버튼 탭하기
Safari 하단 가운데에 있는 공유(↑) 버튼을 탭합니다.
화면 하단 중앙 · 사각형에 화살표 아이콘
3
홈 화면에 추가 선택
공유 시트가 열리면 스크롤해서 "홈 화면에 추가" 항목을 탭합니다.
"홈 화면에 추가" 탭 → 이름 확인 → 추가
4
이름 확인 후 추가
"라이프컬처"로 이름이 설정된 것을 확인하고 우상단 "추가"를 탭합니다.
홈 화면에 라이프컬처 아이콘이 추가됩니다 🎉
안드로이드에서는 Chrome 브라우저로 접속하면 자동 설치 배너가 표시됩니다.
1
Chrome으로 접속
Google Chrome 브라우저로 이 앱에 접속하세요. 잠시 후 화면 하단에 "앱으로 설치하기" 배너가 나타납니다.
Chrome 브라우저로 URL 접속
2
배너에서 "설치" 탭하기
화면 하단에 표시되는 "앱으로 설치하기" 배너에서 "설치" 버튼을 탭합니다.
하단 배너 → 설치 버튼 탭
3
배너가 없다면 메뉴에서 설치
배너가 보이지 않으면 Chrome 우상단 ⋮ 메뉴 → "앱 설치" 또는 "홈 화면에 추가"를 선택하세요.
⋮ 메뉴 → "앱 설치" 또는 "홈 화면에 추가"
4
설치 확인
확인 대화상자에서 "설치"를 탭하면 홈 화면에 라이프컬처 아이콘이 추가됩니다.
홈 화면에 아이콘 추가 완료 🎉
PC에서는 Chrome, Edge 브라우저로 접속하면 주소창에 설치 아이콘이 나타납니다.
1
Chrome / Edge로 접속
Chrome 또는 Microsoft Edge 브라우저로 이 앱에 접속하세요.
Chrome 또는 Edge 브라우저
2
주소창의 설치 아이콘 클릭
주소창 오른쪽에 컴퓨터+다운로드 아이콘(⊕)이 나타나면 클릭합니다.
주소창 우측 설치 아이콘 클릭
3
"설치" 버튼 클릭
팝업에서 "설치"를 클릭하면 앱이 독립 창으로 설치되고 작업 표시줄에 등록됩니다.
데스크탑 앱으로 설치 완료 🎉
