데이터 백업 / 복원

배포(업데이트) 전 필수 절차

  1. 아래 ① 백업 받기로 JSON 파일을 PC에 저장합니다
  2. Publish 탭 → 데이터베이스 재구축으로 배포를 실행합니다
  3. 배포 완료 후 앱을 새로 열고 ② 복원하기에 파일을 업로드합니다
  4. 복원이 끝나면 데이터가 모두 돌아옵니다 ✅

① 데이터 백업

현재 DB에 저장된 모든 데이터를 JSON 파일 1개로 내보냅니다.
배포 전 반드시 실행하세요. 파일은 PC 다운로드 폴더에 저장됩니다.

② 데이터 복원

백업한 JSON 파일을 업로드하면 DB에 자동으로 재삽입합니다.
복원 전 DB가 비어있어야 합니다 (재구축 배포 직후 사용).

주의! 복원은 기존 데이터에 추가됩니다. 이미 데이터가 있는 상태에서 복원하면 중복 데이터가 생깁니다. 반드시 재구축 배포 직후 빈 DB 상태에서만 복원하세요.

클릭하거나 파일을 여기 끌어다 놓으세요
